# FX Travel Jumbo - 开发工具

## 📋 翻译完整性检查工具

### 使用方法

在项目根目录运行:

```bash
node check-translations.js
```

### 功能说明

此工具会自动:

1. **扫描HTML文件** - 找出所有使用 `data-i18n` 属性的元素
2. **提取i18n键** - 列出所有唯一的翻译键
3. **检查完整性** - 验证5种语言(zh-TW, zh-CN, en, ja, ko)是否都有对应的翻译
4. **生成报告** - 显示每个语言的翻译状态和缺失项

### 输出示例

```

============================================================

📋 找到 27 个唯一的i18n键:

   1. nav.about
   2. nav.features
   ...

============================================================

📊 各语言翻译完整性检查:

✅ 🇭🇰 繁體中文 (zh-TW) - 完整
✅ 🇨🇳 简体中文 (zh-CN) - 完整
✅ 🇬🇧 English (en) - 完整
✅ 🇯🇵 日本語 (ja) - 完整
✅ 🇰🇷 한국어 (ko) - 完整

============================================================

🎉 恭喜!所有语言的翻译都完整!

✨ 检查完成!
```

### 何时使用

- ✅ 添加新的UI元素后
- ✅ 修改translations对象后
- ✅ 发布前最终检查
- ✅ 多人协作时确保翻译同步

### 注意事项

- 该工具只检查 `data-i18n` 属性标记的元素
- 动态生成的内容需要手动检查
- Jumbo消息(jumbo.msg.X.greeting/message)由generate-translations.js管理
